.. _glossary:


********
Glossary
********
   
.. glossary::
   
   ``BBox``
        Boundary Box, is the rectangle which enclose content in a PDF page.
        In ``pyxpdf`` a BBox is represented as a 4-int tuple of two diagonal
        vertices of rectangle - (x1, y1, x2, y2)

   ``file-like``
        An object exposing a file-oriented API (with methods such as
        :meth:`read()` or :meth:`write()`) to an underlying resource.

   ``xpdfrc``
        See :doc:`api/config/xpdfrc`
